# Internet of Things - IoT (इंटरनेट ऑफ थिंग्स)

पायथन हार्डवेयर के साथ काम करने और IoT प्रोजेक्ट्स बनाने के लिए बहुत लोकप्रिय है, खासकर Raspberry Pi जैसे छोटे कंप्यूटरों के साथ।

## पायथन और IoT
- **Raspberry Pi**: यह एक क्रेडिट-कार्ड आकार का कंप्यूटर है जिसमें पायथन डिफ़ॉल्ट रूप से इंस्टॉल होता है। आप इसका उपयोग मोटर्स, सेंसर, और लाइट्स को नियंत्रित करने के लिए कर सकते हैं।
- **MicroPython / CircuitPython**: ये पायथन के छोटे संस्करण हैं जो सीधे माइक्रोकंट्रोलर्स (जैसे ESP8266, ESP32, Arduino Nano) पर चलते हैं।

## क्या सीखेंगे?
- सेंसर से डेटा पढ़ना (तापमान, प्रकाश, गति)।
- बाहरी उपकरणों को नियंत्रित करना (LED जलाना, मोटर घुमाना)।
- इंटरनेट के माध्यम से डेटा भेजना (Cloud Integration)।
- स्मार्ट होम डिवाइस बनाना।
