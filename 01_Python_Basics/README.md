# Python Basics (पायथन बेसिक्स)

यह फोल्डर उन लोगों के लिए है जो पायथन प्रोग्रामिंग की शुरुआत कर रहे हैं। यहाँ आप पायथन के मूल सिद्धांत सीखेंगे।

## पाठ्यक्रम (Syllabus)

1. **Introduction (परिचय)**: पायथन क्या है, इसे कैसे इंस्टॉल करें और पहला प्रोग्राम (Hello World) कैसे लिखें।
2. **Variables and Data Types (वैरिएबल और डेटा प्रकार)**: Numbers, Strings, Lists, Tuples, Dictionaries, Sets.
3. **Operators (ऑपरेटर्स)**: Arithmetic, Relational, Logical, Assignment ऑपरेटर्स।
4. **Control Flow (नियंत्रण प्रवाह)**: If-else statements, Loops (for, while).
5. **Functions (फंक्शन्स)**: फंक्शन कैसे बनाएं, पैरामीटर्स, रिटर्न वैल्यू।
6. **File Handling (फ़ाइल हैंडलिंग)**: फ़ाइलों को पढ़ना और लिखना।
7. **Object-Oriented Programming (OOP) - ऑब्जेक्ट-ओरिएंटेड प्रोग्रामिंग**: Classes, Objects, Inheritance, Polymorphism.
8. **Exception Handling (त्रुटि प्रबंधन)**: Try, except, finally ब्लॉक्स।

## उदाहरण (Examples)
नीचे दिए गए `hello.py` फ़ाइल को चलाकर देखें।
