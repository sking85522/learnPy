# App Development (ऐप डेवलपमेंट)

पायथन का उपयोग करके हम डेस्कटॉप और मोबाइल (मल्टीप्लेटफॉर्म) एप्लिकेशन बना सकते हैं।

## प्रमुख टूल्स (Tools)

1. **Tkinter (टिकिन्टर)**: यह पायथन के साथ आने वाली डिफ़ॉल्ट GUI (ग्राफिकल यूजर इंटरफेस) लाइब्रेरी है। सरल डेस्कटॉप ऐप बनाने के लिए अच्छी है।
2. **Kivy (किवी)**: यह एक ओपन-सोर्स पायथन लाइब्रेरी है जो मल्टीटच एप्लिकेशन बनाने के लिए उपयोग होती है। इससे आप Android, iOS, Linux, Windows के लिए ऐप बना सकते हैं (Multiplatform App Development)।
3. **PyQt / PySide**: ये उन्नत डेस्कटॉप एप्लिकेशन बनाने के लिए बहुत शक्तिशाली लाइब्रेरियां हैं।

## Kivy के फायदे
Kivy का सबसे बड़ा फायदा यह है कि आप एक बार कोड लिखते हैं और उसे कंप्यूटर (Windows/Mac) और मोबाइल (Android/iOS) दोनों पर चला सकते हैं।

## उदाहरण
Kivy का एक सरल उदाहरण देखने के लिए `simple_app.py` देखें।
