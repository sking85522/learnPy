# Game Development (गेम डेवलपमेंट)

पायथन में गेम बनाना बहुत मज़ेदार है और प्रोग्रामिंग लॉजिक सीखने का एक शानदार तरीका है।

## प्रमुख लाइब्रेरी: Pygame (पायगेम)
Pygame पायथन मॉड्यूल का एक सेट है जिसे वीडियो गेम लिखने के लिए डिज़ाइन किया गया है। यह आपको ग्राफ़िक्स बनाने, ध्वनियां बजाने और कीबोर्ड/माउस इनपुट को संभालने की सुविधा देता है।

## गेम डेवलपमेंट के चरण
1. **गेम लूप (Game Loop)**: वह लूप जो गेम को चालू रखता है (स्क्रीन अपडेट करना, इवेंट्स चेक करना)।
2. **इवेंट्स (Events)**: उपयोगकर्ता क्या कर रहा है? (जैसे तीर कुंजियाँ दबाना)।
3. **अपडेट (Update)**: इवेंट्स के आधार पर गेम की स्थिति (जैसे खिलाड़ी की स्थिति) बदलना।
4. **रेंडर (Render)**: स्क्रीन पर सब कुछ फिर से बनाना।

## शुरुआत
2D गेम्स बनाने के लिए Pygame सबसे अच्छा शुरुआती बिंदु है।
