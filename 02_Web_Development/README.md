# Web Development (वेब डेवलपमेंट)

पायथन का उपयोग करके शक्तिशाली वेब एप्लिकेशन कैसे बनाएं, यह हम यहाँ सीखेंगे। हम मुख्य रूप से Full-stack विकास पर ध्यान देंगे।

## प्रमुख फ्रेमवर्क्स (Popular Frameworks)

1. **Django (जैंगो)**: यह एक उच्च-स्तरीय (high-level) पायथन वेब फ्रेमवर्क है जो तेजी से विकास और स्वच्छ, व्यावहारिक डिजाइन को प्रोत्साहित करता है। यह बड़े और जटिल वेब एप्लिकेशन बनाने के लिए बहुत अच्छा है।
2. **Flask (फ्लास्क)**: यह एक माइक्रो-फ्रेमवर्क है। यह हल्का और लचीला है, जो छोटे से मध्यम आकार के प्रोजेक्ट्स के लिए उपयुक्त है।
3. **FastAPI (फास्ट-एपीआई)**: यह आधुनिक, तेज (उच्च-प्रदर्शन) वेब फ्रेमवर्क है जो मानक पायथन टाइप हिंट्स के आधार पर API बनाने के लिए है।

## क्या सीखेंगे? (What will you learn?)
- **Backend (बैकएंड)**: सर्वर-साइड लॉजिक, डेटाबेस कनेक्टिविटी (SQL/NoSQL), API बनाना।
- **Frontend (फ्रंटएंड)**: HTML, CSS, JavaScript के साथ पायथन टेम्प्लेट (Jinja2) का एकीकरण।
- **Deployment (डिप्लॉयमेंट)**: अपने वेब ऐप को इंटरनेट पर लाइव कैसे करें (उदा. Heroku, AWS, PythonAnywhere)।

## शुरुआत कैसे करें?
एक साधारण फ्लास्क ऐप बनाने के लिए `simple_web_app.py` देखें।
