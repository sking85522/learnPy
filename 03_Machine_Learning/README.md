# Machine Learning (मशीन लर्निंग)

मशीन लर्निंग (ML) कंप्यूटर को स्पष्ट रूप से प्रोग्राम किए बिना डेटा से सीखने की क्षमता प्रदान करता है। पायथन ML के लिए सबसे लोकप्रिय भाषा है।

## महत्वपूर्ण लाइब्रेरियां (Important Libraries)

1. **NumPy (नम-पाई)**: वैज्ञानिक कंप्यूटिंग और सरणियों (arrays) के साथ काम करने के लिए।
2. **Pandas (पांडास)**: डेटा विश्लेषण और हेरफेर (manipulation) के लिए।
3. **Matplotlib & Seaborn (मैटप्लॉटलिब और सीबॉर्न)**: डेटा विज़ुअलाइज़ेशन (ग्राफ़ बनाने) के लिए।
4. **Scikit-Learn (सायकिट-लर्न)**: क्लासिक मशीन लर्निंग एल्गोरिदम (जैसे रिग्रेशन, क्लासिफिकेशन, क्लस्टरिंग) के लिए।

## मशीन लर्निंग के प्रकार (Types of ML)
- **Supervised Learning (सुपरवाइज्ड लर्निंग)**: लेबल किए गए डेटा से सीखना। (उदा. कीमत की भविष्यवाणी करना)।
- **Unsupervised Learning (अनसुपरवाइज्ड लर्निंग)**: बिना लेबल वाले डेटा में छिपे हुए पैटर्न खोजना। (उदा. ग्राहकों को समूहों में बांटना)।
- **Reinforcement Learning (रीइन्फोर्समेंट लर्निंग)**: परीक्षण और त्रुटि (trial and error) के माध्यम से सीखना।
