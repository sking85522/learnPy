# 🐍 सम्पूर्ण पायथन मास्टरक्लास (Complete Python Masterclass)

नमस्ते! इस रिपॉजिटरी में आपका स्वागत है। यह रिपॉजिटरी आपको **पायथन** में शुरुआत से लेकर एक्सपर्ट (Master) स्तर तक ले जाने के लिए डिज़ाइन की गई है। यहाँ उपलब्ध सभी अध्ययन सामग्री (Learning Materials) **हिंदी** में है ताकि आप आसानी से समझ सकें।

## 🌟 हम क्या क्या सीखेंगे?

इस रिपॉजिटरी को अलग-अलग फोल्डर्स में बांटा गया है, जिनमें निम्नलिखित विषय शामिल हैं:

1. **[01_Python_Basics](./01_Python_Basics/)**: पायथन की एबीसी (Variables, Loops, Functions, OOP)।
2. **[02_Web_Development](./02_Web_Development/)**: फुल-स्टैक वेब डेवलपमेंट (Django, Flask, FastAPI)।
3. **[03_Machine_Learning](./03_Machine_Learning/)**: डेटा से सीखना (Pandas, Scikit-Learn)।
4. **[04_AI_and_Neural_Networks](./04_AI_and_Neural_Networks/)**: कस्टम AI, न्यूरल नेटवर्क, डीप लर्निंग (TensorFlow)।
5. **[05_App_Development](./05_App_Development/)**: मल्टीप्लेटफॉर्म ऐप डेवलपमेंट (Kivy)। मोबाइल और कंप्यूटर दोनों के लिए ऐप बनाना।
6. **[06_IoT](./06_IoT/)**: पायथन के साथ हार्डवेयर (Raspberry Pi, MicroPython) को नियंत्रित करना।
7. **[07_Game_Development](./07_Game_Development/)**: Pygame का उपयोग करके मजेदार गेम बनाना।
8. **[08_Computer_Vision_and_OCR](./08_Computer_Vision_and_OCR/)**: लाइव कैमरा (वेबकैम) से डेटा लेना, चेहरे (Face) पहचानना, और इमेज/वीडियो से अन्य ऑब्जेक्ट्स या टेक्स्ट (OCR) की पहचान करना।
9. **[Projects](./Projects/)**: एक विशेष प्रोजेक्ट फोल्डर जहाँ हम सीखे गए सभी कौशल को मिलाकर वास्तविक दुनिया के प्रोजेक्ट (Special Projects) बनाएंगे।

## 🚀 इसका उपयोग कैसे करें?

1. यदि आप बिलकुल नए हैं, तो **01_Python_Basics** से शुरू करें।
2. **सॉफ्टवेयर इंस्टॉल करें**:
   - सबसे पहले Python इंस्टॉल करें (python.org से)।
   - फिर सभी ज़रूरी लाइब्रेरियां इंस्टॉल करने के लिए टर्मिनल/कमांड प्रॉम्प्ट में यह कमांड चलाएं:
     `pip install -r requirements.txt`
3. हर फोल्डर में एक `README.md` फ़ाइल है। उसे खोलें और उसमें दिए गए निर्देशों और थ्योरी को पढ़ें।
4. वहां दिए गए उदाहरण कोड (`.py` फाइलों) को अपने कंप्यूटर पर चला कर देखें।
5. जब आप बुनियादी चीजें समझ लें, तो अपनी रुचि के अनुसार वेब, AI या गेम डेवलपमेंट वाले फोल्डर में जाएं।
5. अंत में, **Projects** फोल्डर में जाएं और खुद से कुछ नया बनाने की कोशिश करें!
